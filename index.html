Comment faire ? (Tutoriel)

1. Récupération : Cliquez sur l'un des boutons ci-dessus pour copier les données dans votre presse-papier.

2. Dans EDT (Élèves) : Allez dans l'onglet Emploi du temps > menu Élèves > affichage Liste (le 1er en haut à gauche).

3. Importation : Faites Ctrl + V. Une fenêtre d'importation s'ouvre. Vous verrez apparaître les noms des élèves et leurs groupes.

4. Mapping : Au-dessus des colonnes, choisissez l'item correspondant dans le menu déroulant.

5. Matières : Pour les matières, la procédure est la même, mais placez-vous dans l'onglet Matières d'EDT.

📚 Comment utiliser le Planning Automatique ?

Pour utiliser le mode Planning automatique, vous devez réaliser les actions suivantes :

  • Etape 1 : Dans EDT, allez dans Gestion par semaine > Classes. Il faut annuler manuellement tous les cours des classes concernées par l'examen.
  • Etape 2 : Dans EDT, allez dans Gestion par semaine > Cours (premier sous-menu "Afficher la liste"). Sélectionnez la/les semaines concernées, cochez "Cours annuels et ponctuels" puis faites un Copier (ou bouton copier/coller).
  • Etape 3 : Créez un nouveau fichier Excel et collez les données issues d'EDT. Vous devrez créer 2 fichiers : 1 premier avec tous les cours annulés pour le ${getExamTitle()} (mais pas les cours annulés des professeurs absents) et 1 second fichier avec ous les cours maintenus. Enregistrez vos 2 fichiers.
  • Etape 4 : Importez les fichiers dans le logiciel via les boutons "Parcourir".
  • Etape 5 : Cliquez sur le bouton "⚡ Planning Automatique".
